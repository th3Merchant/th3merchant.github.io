Calmer cleaning

Use it wisely

Start low, keep it clean, and follow the product instructions.

For infants and newborns, start with the lowest suction level and increase only if needed. The device is not waterproof, so removable parts should be cleaned separately and the main body should stay dry.

  • Use the lowest suction first
  • Do not immerse the device body
  • Clean and dry parts after use
  • Use a 5V adapter for charging
